# Interaction Model

## Fluxo do CRE
1. Input humano
2. Context packet explicito
3. Classificacao da rejeicao automatica
4. Identificacao do gatilho
5. Modelagem do contexto de entrada
6. Simulacao no Laboratorio Virtual
7. Coleta de evidencias
8. Reforco ou ajuste da estrategia

## Fluxo operacional do Laboratorio Virtual
1. Definir cenario
2. Declarar hipotese
3. Aplicar estrategia de reframing
4. Registrar resultado observado
5. Comparar com a hipotese
6. Registrar aprendizado

## Regras
- O fluxo precisa manter rastreabilidade
- Toda execucao deve ser explicita
- Toda conclusao precisa de evidencia
- O laboratorio nao substitui a validação etica
